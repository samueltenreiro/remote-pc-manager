# 🖥️ Remote PC Manager

A Python-based remote administration tool that allows users to control and monitor a Windows computer through a Telegram bot.

## 🚀 Features

- 📁 Remote file management
- 📸 Screenshot capture
- 📷 Webcam capture
- 💻 System monitoring
- 📶 Wi-Fi management
- 🎵 Media controls
- 📄 Document management
- ⏰ Reminders and automation
- 🌦️ Weather information
- 🔍 Web search utilities
- ⚙️ Remote application control

## 🛠️ Technologies

- Python
- Telegram Bot API
- OpenCV
- Pillow
- PyAutoGUI
- psutil
- NumPy
- Requests
- JSON

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/samueltenreiro/remote-pc-manager.git
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```

## ⚠️ Configuration

Before running the project, you must configure:

- Your Telegram Bot Token
- Any required API keys
- Project configuration files

Sensitive information is **not included** in this repository for security reasons.
